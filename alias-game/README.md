WORD GAMES
A collection of two party games: Alias (word describing) and Pictionary
(word drawing). Built with React and Tailwind CSS.

/************************Alias (main branch)***********************************/
Classic word-guessing game where players describe words without saying them.

Features:
-200 Russian words across multiple categories
-Adjustable words per card (1-5)
-Manual word entry option
-Arrow key navigation

How to Play:
-Teams take turns
-One player describes the word without saying it
-Teammates guess


/*****************Pictionary (pictionary branch)*******************************/
Drawing game where players illustrate words for their team to guess.

Features:
-200 Russian words for drawing
-Built-in timer (30-120 seconds)
-Score tracking for 2 teams
-Skip button for difficult words

How to Play:
-Teams alternate turns
-One player draws the word (no talking!)
-Team guesses while timer runs
-Click "Correct!" for points or "Skip" to pass

SWITCHING BETWEEN GAMES
# Play Alias (word describing game)
git checkout main

# Play Pictionary (word drawing game)
git checkout pictionary
